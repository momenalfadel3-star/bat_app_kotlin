# الخفاش - تطبيق أندرويد

تطبيق أندرويد مكتوب بلغة **Kotlin** - نسخة محولة من المشروع الأصلي Python/Kivy.

## المميزات

- واجهة مستخدم داكنة احترافية
- دعم HTTP و HTTPS
- خيوط متعددة قابلة للتخصيص
- حفظ الهوستات تلقائياً
- خدمة خلفية مستمرة
- إحصائيات مباشرة للطلبات
- سجل أحداث مباشر

## المتطلبات

- Android 5.0+ (API 21)
- إذن الإنترنت

## البناء

```bash
./gradlew assembleDebug
```
